# RobotAI Website

A modern, professional website for RobotAI - an AI and robotics innovation company.

## Project Structure

```
robomai.github.io/
├── index.html              # Home page
├── pages/
│   ├── about.html         # About Us page
│   ├── services.html      # Services page
│   ├── portfolio.html     # Portfolio/Projects page
│   ├── team.html          # Team page
│   ├── blog.html          # Blog page
│   └── contact.html       # Contact page
├── css/
│   └── style.css          # Main stylesheet
├── js/
│   └── main.js            # JavaScript functionality
└── README.md              # This file
```

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Navigation Menu**: Sticky header with mobile hamburger menu
- **Multiple Pages**: Home, About, Services, Portfolio, Team, Blog, and Contact pages
- **Interactive Elements**: Form submissions, smooth scrolling, and hover effects
- **Performance Optimized**: Fast loading and smooth interactions

## Pages

### Home (index.html)
Features a hero section, services overview, featured projects, and call-to-action.

### About (pages/about.html)
Company story, mission, vision, core values, and key statistics.

### Services (pages/services.html)
Detailed information about AI Solutions, Robotics Engineering, Data Analytics, and Custom Development.

### Portfolio (pages/portfolio.html)
Showcase of completed projects with case studies.

### Team (pages/team.html)
Meet the team members with their roles and social links.

### Blog (pages/blog.html)
Blog posts about AI, robotics, and technology trends.

### Contact (pages/contact.html)
Contact form and company information.

## Customization

### Images and Videos
Placeholders are ready to be replaced with actual images and videos:
- Update `.placeholder` divs with actual `<img>` and `<video>` tags
- Replace Hero Image placeholders
- Add project images
- Add team member avatars
- Add blog post images

### Colors
Edit the CSS variables in `css/style.css`:
```css
:root {
    --primary-color: #1a3a52;      /* Dark blue */
    --secondary-color: #0066cc;    /* Bright blue */
    --accent-color: #ff6b35;       /* Orange */
    --light-bg: #f5f5f5;           /* Light gray */
    --dark-text: #333333;          /* Dark text */
    --light-text: #666666;         /* Light text */
}
```

### Content
Edit the HTML files to update:
- Company name and description
- Service descriptions
- Team member information
- Blog posts
- Contact information

## Getting Started

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Replace placeholder images and videos with your content
4. Update all text content with your company information
5. Customize colors and styling as needed
