# Hardware Overview

## Camera Components

The RobotAI Camera system consists of several integrated components working together to provide high-precision imaging.

### Main Sensor Array

The camera features:
- **Resolution:** 2048 × 1536 pixels
- **Sensor Type:** RGB + Depth sensor
- **Frame Rate:** Up to 30 fps
- **Depth Range:** 300mm - 1500mm
- **Field of View:** 70° horizontal, 53° vertical

### Optical System

- **Lens Type:** Fixed focus wide-angle
- **Aperture:** f/2.8
- **Image Stabilization:** Optical (6-axis)
- **Autofocus:** Contrast-based continuous autofocus

### LED Indicator

The LED status light provides real-time camera status:

| Color | State | Meaning |
|-------|-------|---------|
| Red | Solid | Error or offline |
| Green | Solid | Connected and ready |
| Green | Pulsing | Recording or processing |
| Blue | Blinking | Firmware update in progress |
| Yellow | Blinking | Low power or thermal warning |

### Dimensions & Weight

- **Dimensions:** 85mm (W) × 65mm (H) × 95mm (D)
- **Weight:** 250g
- **Operating Temperature:** 0°C to 40°C
- **Storage Temperature:** -10°C to 50°C

## Connectivity

### USB Interface

- **Standard:** USB 3.0 (SuperSpeed)
- **Data Transfer Rate:** Up to 400 MB/s
- **Power Over USB:** Supported (optional external power)
- **Backward Compatible:** USB 2.0 (reduced performance)

### Power Input

- **Input Voltage:** 5V DC
- **Input Current:** 2.0A
- **Connector Type:** Micro USB
- **Alternative:** USB bus power (when supported)

## Thermal Management

The camera includes:
- Advanced heat dissipation design
- Passive cooling vents
- Thermal monitoring system
- Automatic throttling for overheating protection

## Environmental Specifications

| Parameter | Value |
|-----------|-------|
| Operating Humidity | 10% - 85% (non-condensing) |
| Storage Humidity | 5% - 95% (non-condensing) |
| Dust Protection | IP54 rating |
| Vibration Resistance | Up to 2G |
| Impact Resistance | Drop tested from 1.5m |

## Mounting Options

The camera supports multiple mounting configurations:

- **Standard Mount:** M4 tripod socket on base
- **Quick Release:** 1/4"-20 UNC thread
- **Wall Mount:** Bracket included
- **Robotic Arm Mount:** Adapter available (sold separately)

## Mechanical Interface

- **Tripod Mount:** 1/4"-20 UNC (standard)
- **Bracket Compatibility:** Arca-Swiss compatible
- **Cable Management:** Built-in cable clips
